# Radar Simulator

Проект производственной практики (НИР).  
**Цель:** реализация имитатора радиолокационной развёртки и вычисление параметров радара.

## Структура проекта
- `core/` – ядро расчётов (RadarCalculator)
- `src/` – исходный код приложения (main, mainwindow)
- `ui/` – файлы интерфейса Qt (mainwindow.ui)

## Сборка
Требования: **Qt5**, **CMake**.  
mkdir build && cd build
cmake ..
make

## Режимы работы
- Временной анализ – проверка возможности реализации по времени облучения, расчёт максимальной дальности.
- Мощностной анализ – определение максимальной дальности цели с ЭПР 10 м².
- Комбинированный анализ – сравнение временных и мощностных ограничений, расчёт количества импульсов в пачке.

## Ветки Git
- main – стабильные версии проекта
- dev – ежедневная разработка
- testing – версии для тестирования

## История версий

### v0.1 (текущая)

- Реализованы три режима расчёта характеристик РЛС.
- Поддержка двух типов антенн: щелевая и зеркальная.
- Поддержка двух типов зондирующего сигнала: импульсный и квазинепрерывный.
- Разделение исходных файлов по папкам src, core, ui.
- Настроена сборка через CMake.
