# Radar Simulator

Проект производственной практики.

## Цель проекта

Разработка программы расчёта характеристик радиолокационной станции и имитации радиолокационной развертки.

## Используемые технологии

- C++
- Qt5
- Git

## Структура веток

- main — стабильные версии
- dev — ежедневная разработка
- testing — версии для тестирования

## План разработки

### Неделя 1–2

- Расчёт характеристик РЛС
- Временной анализ
- Мощностной анализ
- Суммарный анализ

### Следующий этап

- Имитация радиолокационной развертки
- Отображение целей
- Таблица целей
- Движущиеся цели
